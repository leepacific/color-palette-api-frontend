# Component Inventory — color-palette-api frontend

**Source**: `context/lead-reports/lab/component-inventory-report.md`

Total: **23 components** (4 layout, 8 interactive, 6 data-display, 3 overlay, 2 utility).
All data-dependent components have 4 states designed (§2.3 compliance).
All interactive elements have 4 states designed (§2.4 compliance).

## Types (shared across components)

```ts
import type { components } from '@/api.d';  // generated by openapi-typescript from api-contract.yaml

type Color = {
  hex: string;           // "#D00000"
  rgb: { r: number; g: number; b: number };
  hsl: { h: number; s: number; l: number };
  oklch: { l: number; c: number; h: number };
  name?: string;         // "Red" (from backend)
};

type PaletteResource = components['schemas']['Resource_PaletteBody'];
type ContrastMatrixResource = components['schemas']['Resource_ContrastMatrixBody'];
type PaletteExplanationResource = components['schemas']['Resource_PaletteExplanationBody'];
type CodeExportResponse = components['schemas']['Resource_CodeExportBody'];
type SemanticTokenBundle = components['schemas']['ExtendedSemantic'];

type ColorblindMode =
  | 'none' | 'protanopia' | 'deuteranopia' | 'tritanopia'
  | 'protanomaly' | 'deuteranomaly' | 'tritanomaly'
  | 'achromatopsia' | 'achromatomaly';

type CodeExportFormat =
  | 'tailwind-config' | 'css-vars-hex' | 'css-vars-oklch' | 'shadcn-globals'
  | 'scss' | 'mui-palette' | 'swift-uicolor' | 'android-xml' | 'dtcg-json';
```

## C1. `<ColorSwatch>` — 4 states (data + interactive)

```ts
interface ColorSwatchProps {
  color: Color;
  locked: boolean;
  index: number;
  focused: boolean;
  onLock: () => void;
  onCopy: (notation: 'hex' | 'oklch' | 'hsl') => void;
}
```

**Visual**:
- Large color fill top (flex-1)
- Bottom strip (`bg-raised`): hex (`text-3xl`), oklch (`text-sm`), hsl (`text-xs`), all monospace, all click-to-copy
- Lock badge `[L]` top-right when locked

**States** (interactive §2.4):
- Default: solid fill, no border
- Hover: 1px `--border-accent` inner ring; hex gains 1px underline
- Active: ring thickens to 2px; `scale(0.99)`
- Focus-visible: 2px `--border-accent` outer ring (2px offset)
- Locked: `[L]` badge persistent + dim ring
- Copy feedback: 120ms background flash on copied notation

## D1. `<PaletteDisplay>` — 4 states (data §2.3)

```ts
interface PaletteDisplayProps {
  palette: PaletteResource | null;
  locked: number[];
  focusedIndex: number | null;
  state: 'default' | 'loading' | 'error';
  error?: { type: string; code: string; message: string; requestId: string };
  onLock: (idx: number) => void;
  onFocus: (idx: number) => void;
  onCopy: (idx: number, notation: 'hex' | 'oklch' | 'hsl') => void;
}
```

Renders N `<ColorSwatch>` (5 by default) in horizontal flex row. Composite score displayed below.

**Data states**:
- **Default**: 5 swatches + composite score visible
- **Empty**: `no palette · press [R] to generate` (if null) in `--fg-tertiary`
- **Loading**: 5 `--bg-raised` blocks each with a blinking caret (synchronized) — NOT pulsing skeleton
- **Error**: `████` fills in `--semantic-error` dim + error line below: `error: ${error.type} (${error.code}) · [r] retry · [c] copy requestId · [s] reset`

## D2. `<JsonSidebar>` — 4 states
